Broadsoft Technologies Walk-in For Testing Engineer Posted Date: 29/07/2016

Job Description

 Job Description:
Minimum 2+ years testing experience in Security Testing and Load testing on Web Services using manual or automated methodologies Candidate with telecom domain exposure is preferred.

Skills:
Development of a scalable performance and load test framework.
Construct and maintain performance test scripts for web-based applications using JMeter.
Experience with OWASP testing Guide / Open Source Security.
Testing Methodology - Manual.
Experience with web application vulnerability scanning tools (Metasploit , Burpsuite, backtrack, Nessus, nmap, Zed Attack Proxy).
Wireshark and tools available on Kali Linux.
Valid Certification - CEH, ECSA, ECSP, Sans, GIAC Certification, and CISSP.
Use of Issue tracking systems like Jira.
Excellent communication and reporting skills.
Basic Linux knowledge is mandatory.
